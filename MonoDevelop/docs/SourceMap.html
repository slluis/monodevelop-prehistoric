Introduction

Sometimes it can be hard to find your way around the MonoDevelop source tree. This is intended as a brief description on where and what things are, and a couple tips to find things.

Map

MonoDevelop.Core - contains the bare necessities for the IDE. It has no external dependencies and is located at src/Libraries/MonoDevelop.Core/

MonoDevelop.Base - contains the base of the IDE. It begins to depend on Gtk/GNOME at this point and is located at src/Main/Base/

AddIns - various plugins that add to the IDE's functionality are located at src/AddIns/ See also the AddIns guide.

Last resort

Sometimes its still can be confusing so running something like the two below commands can be used as a last resort to find things.

find . -name '*TypeName*.cs'

find . -name '*.cs' | xargs grep TypeName
